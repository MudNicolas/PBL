# 获取源文件后

-   `cd server` 进入服务端文件夹
-   `npm install` 安装依赖
-   `node/nodemon/supervisor app.js` 开启服务/自动重启服务
-   开启另一个终端
-   `cd vue-element-admin`
-   `npm install` 安装依赖
-   `npm run dev` 开启在 develop 模式下开启前端服务器
